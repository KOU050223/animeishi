# Animeishi プロジェクト改善・グロース戦略一覧

## 1. 検索精度の向上 (Search Experience)
* **「揺らぎ」の許容と正規化**: ひらがな・カタカナ・英数字・大文字小文字を正規化してマッチング。
* **略称カラムの追加**: 「ダンまち」「転スラ」などの一般的な略称でもヒットするよう、タイトルマスタに検索用タグを持たせる。
* **クライアントサイド・ファジーサーチ**: `Fuse.js` 等を活用し、多少の打ち間違いがあっても候補を表示する。
* **トレンドブースト**: 全体検索結果の中で、「今期の人気作」や「ユーザーの視聴傾向に近いジャンル」を上位に表示させる。

## 2. 共有・OGP戦略 (Social Visibility)
* **動的OGP生成**: Cloudflare Workers（Satori/resvg）を活用し、視聴履歴や分析結果を反映した「世界に一つだけの名刺画像」をエッジで生成。
* **属性の可視化**: Geminiによる分析結果（「作画オタク」「泣きゲー系」など）をバッジやグラフとして画像に埋め込む。
* **シェア専用導線の設計**: 分析完了画面に「X（旧Twitter）に名刺を公開する」ボタンを配置し、拡散を促す。

## 3. UI/UX向上のコツ (Design Quality)
* **スケルトンスクリーンの導入**: データ読み込み中に枠組みを表示し、ユーザーの待機ストレスを軽減（体感速度の向上）。
* **マイクロインタラクション**: チェックを入れた際のアニメーションや、スワイプ時の心地よい挙動（Reanimated等を使用）を徹底する。
* **ダークモードの最適化**: 深夜の視聴管理を想定し、コントラストを抑えた「目に優しいダークモード」を用意。
* **情報の「密度」と「余白」**: 多くのタイトルを並べつつも、タップのしやすさや視認性を損なわない余白設計。

## 4. プロモーション・流行らせ方 (Growth)
* **クールごとの振り返りイベント**: クール終了時に「今期のマイベスト名刺」を生成するよう促し、共通ハッシュタグで盛り上げる。
* **「物理」名刺としての活用**: オフ会やイベント等で、QRコード画面を提示して交換する「デジタル名刺」としての体験を強調。
* **ニッチ・インフルエンサーへの提案**: アニメ感想を投稿している小〜中規模のユーザーに対し、リスト管理ツールとして提案し、草の根的に広める。

## 5. ストア公開のコツ (ASO & Conversion)
* **ベネフィットを語るスクリーンショット**: 「リストを作れる」ではなく、「一瞬で自分だけのアニメ名刺が完成」「AIが好みを分析」など、得られる体験を最初の3枚で伝える。
* **ASOキーワードの最適化**: 「アニメ」「リスト」「共有」「記録」「感想」「視聴管理」など、検索ボリュームのある単語をタイトルや説明文に組み込む。
* **15秒のプレビュー動画**: QRスキャンの速さや、名刺一覧が滑らかに動く様子を動画で見せ、アプリの信頼性と品質をアピール。
